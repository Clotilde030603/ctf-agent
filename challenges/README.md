# Challenges Workspace

Put each challenge in its own subdirectory.

Example:

```text
challenges/example/
  README.md
  notes.md
  files/
  solve.py
  artifacts/
```

Recommended practice:

- Keep original files under `files/`.
- Keep working notes in `notes.md`.
- Put solver scripts in the challenge root.
- Put large generated outputs in `artifacts/`.
